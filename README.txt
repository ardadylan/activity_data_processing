# Activity Data Processing

## Overview
This Python script processes activity data stored in CSV files. 
It aggregates daily metrics such as average resting rate (rr), average heart rate (hr), and total calories burned.
The script reads all CSV files from a specified directory, processes them, and saves the aggregated results into new CSV files.

## Features
- Reads multiple CSV files from a specified directory.
- Processes each file to calculate daily averages and sums for selected metrics.
- Outputs aggregated data into new CSV files.

## Requirements
- Python 3.x
- Pandas
- NumPy

## Installation
1. Clone or download this repository.
2. Ensure you have Python 3.x installed on your machine.
3. Install the required packages using pip in the terminal:
   pip install pandas numpy
